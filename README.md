
### Prerequisites

1. Install nodeJs
2. Install npm on local device


### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/varun-saini-18/bug-tracker.git
   ```
2. Install NPM packages
   ```sh
   npm install
   
   ```
   
3. Open first terminal and run command
```JS
  node app.js
   ```
